# 📐 Calculadora de Áreas em Portugol

Este projeto é uma calculadora simples desenvolvida em **Portugol**, capaz de calcular a área de diferentes formas geométricas.

## 🚀 Funcionalidades

- 🔺 Área do triângulo
- ▭ Área do retângulo
- ◼️ Área do quadrado
- ⚪ Área do círculo

## 🛠️ Tecnologias utilizadas

- Portugol (Portugol Studio)

## ▶️ Como executar

1. Abra o código no **Portugol Studio**
2. Execute o programa
3. Escolha uma das opções do menu:
   - 1 → Triângulo
   - 2 → Retângulo
   - 3 → Quadrado
   - 4 → Círculo
4. Insira os valores solicitados
5. O programa exibirá a área calculada
